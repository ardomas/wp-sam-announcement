== Description ==

This plugin allows administrators to create and display announcements easily from the WordPress dashboard.  
It includes custom post types, templates, and shortcode support for displaying announcements across your site.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/your-plugin-name` directory, or install through the WordPress admin panel.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Announcements** in your dashboard to start creating posts.

== Frequently Asked Questions ==

= Can I customize the announcement template? =
Yes. You can override the default template by copying `single-announcement.php` into your theme directory.

= Does it support Gutenberg or Classic Editor? =
Yes, both editors are supported.
